# User registration application 

##  Prerequisite
Make sure your computer has the following:

* Node.js version 20 or higher

## Instructions

1. clone the repository 
    ```
   https://github.com/Elie-MK/User_frontend.git
    ```
2. Install dependencies
     ```
   npm install
    ```
3. Run the project 

    ```
   npm run dev
    ```
4. Build project 
    ```
   npm run build
    ```
